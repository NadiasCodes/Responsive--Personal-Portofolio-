Achievements

SheCodes Basics

SheCodes Basics is the perfect first dive into front-end development. It teaches you how to build a page in HTML, how to style your page and target specific elements in CSS and how to animate it with JavaScript. By the end of this workshop, it equips you with skills in order to build a simple interactive landing page.

SheCodes Basics Add-on

SheCodes Basics Add-on continues your introduction to front-end development and development tools. It teaches new techniques in HTML, CSS and JavaScript to build on the knowledge based in SheCodes Basic.

SheCodes Plus

SheCodes Plus is ideal for those looking to transition to a career in tech or advance in their current career. It teaches advanced page animation techniques with JavaScript functions, how to integrate APIs into your code to fetch real data and how to use code management tools that professionals use. This workshop could be especially helpful for Product/Project managers, UX/UI designers, Customer support, and QA testers to name a few.

SheCodes Plus Add-on

SheCodes Plus Add-on teaches you the potential of advanced Javascript, allowing you to build more sophisticated interactions and features into your projects. This add-on is ideal for SheCodes Plus graduates who would like to build on the Javascript skills they already learned in Plus.

SheCodes AI Add-on

SheCodes AI Add-on will show you the potential of Artificial Intelligence and how to build a text-generating tool powered by AI. You’ll build on your HTML, CSS and JavaScript knowledge and learn to use an AI API in your code. This add-on is great for anyone looking to learn how to turn AI into a cool feature that users will love. It’s especially relevant for tech professionals working with AI-powered products or tools, and any aspiring developer!

SheCodes Responsive

SheCodes Responsive is ideal for those looking to become professional developers. It teaches you how to make your code responsive to any device and how to build a multi-page website.

SheCodes Responsive Add-on

SheCodes Responsive Add-on is ideal for anyone looking to build on the coding knowledge learned with Responsive. Get creative with your coding skills and add more sophistication to your responsive design using more advanced CSS techniques.

SheCodes React

SheCodes React is ideal for those looking to become professional developers. You will become familiar with the high-in-demand React framework.

SheCodes React Add-on

SheCodes React Add-on is a great choice for React graduates who want to build more advanced knowledge after the workshop. It’s ideal for anyone who would like to work towards becoming a professional developer and gain more advanced knowledge of one of the most in-demand coding frameworks today.
